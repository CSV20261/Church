# Color Normalization Map

## Primary Buttons (Call-to-Action)
- `bg-blue-600` → `bg-primary-600`
- `bg-blue-700` → `bg-primary-700`
- `hover:bg-blue-700` → `hover:bg-primary-700`
- `hover:bg-blue-800` → `hover:bg-primary-800`
- `text-blue-600` → `text-primary-600`
- `text-blue-700` → `text-primary-700`
- `ring-blue-500` → `ring-primary-500`
- `border-blue-600` → `border-primary-600`

## Success States (Keep green for semantic meaning)
- `bg-green-600` → `bg-primary-600` (for primary actions)
- `bg-green-700` → `bg-primary-700`
- `text-green-600` → `text-primary-600` (for primary text)
- `text-green-700` → `text-primary-700`
- Keep `bg-green-100`, `text-green-700` for success badges/states

## Neutral/Gray (Standardize to neutral palette)
- `bg-gray-` → `bg-neutral-`
- `text-gray-` → `text-neutral-`
- `border-gray-` → `border-neutral-`

## Backgrounds
- `bg-gradient-to-br from-green-50 via-white to-blue-50` → `bg-neutral-50`
- Random gradients → solid neutral backgrounds

## Focus States
- `focus:ring-blue-500` → `focus:ring-primary-500`
- `focus:ring-green-500` → `focus:ring-primary-500`
